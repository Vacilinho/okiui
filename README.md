# okiui

Design System for React

## Packages

| Package         | Description                                 | Version |
| --------------- | ------------------------------------------- | ------- |
| `@okiui/tokens` | Design tokens (colors, typography, spacing) | 0.1.0   |

## Quick Start

### Install

npm install @okiui/tokens

### Use tokens in TypeScript

import { colors, typography, spacing } from '@okiui/tokens'

### Use tokens as CSS variables

@import '@okiui/tokens/css';

Then use: var(--oki-colors-primary-500)

## Philosophy

okiui is opinionated. It's not trying to be everything for everyone.
It's a design system for interfaces that need personality — bold colors,
expressive typography, and intentional motion.
